PlantGuard AI is an end-to-end deep learning application built to detect plant diseases from leaf images in real time. The model is a custom 4-block VGG-style CNN trained on the PlantVillage dataset, achieving 92% classification accuracy across 39 disease categories.

Built as a portfolio project demonstrating skills in deep learning, model deployment, and full-stack web development. Open to Data Analyst and Data Scientist roles.
